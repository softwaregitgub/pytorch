# pytorch

1.标量由只有一个元素的张量表示

>>> import torch
>>> x = torch.tensor([3.0])
>>> y = torch.tensor([2.0])
>>> x + y, x * y, x / y, x**y
(tensor([5.]), tensor([6.]), tensor([1.5000]), tensor([9.]))


2.你可以将向量视为标量值组成的列表
>>> x = torch.arange(4)
>>> x
tensor([0, 1, 2, 3])

3.通过张量的索引来访问任一元素
tensor([0, 1, 2, 3])
>>> x[3]
tensor(3)

4.访问张量的长度

>>> len(x)
5.只有一个轴的张量，形状只有一个元素
>>> x.shape
torch.Size([4])
